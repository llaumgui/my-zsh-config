## Vagrant aliases

Plugin provided aliases for [Vagrant](http://www.vagrantup.com/). 

### Alias

* __vup__: Start the box.

~~~
vagrant up
~~~

* __vdown__: Halt the box.

~~~
vagrant halt
~~~

* __vssh__: SSH connection to the box.

~~~
vagrant ssh
~~~

* __vsync__: Rsync between host and guest box .

~~~
vagrant rsync-auto
~~~
